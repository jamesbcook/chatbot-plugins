# Help Plugin


## Details

* This plugin will show the help menu from all the plugins running

```
CMD: /help
Help: /help this message
```

### Example

```
/help
-------------------------
/crypto {cryptocurrency}
/giphy {string}
/help this message
/hibp-email {email}
/hibp-password {passsword}
/media {url}
/reddit {subreddit}
/weather {city}
```
